# PROGETTO-Biblioteca che gestisce i manoscritti eventuali trascrizioni e
caricamento delle immagini.
Un manager all interno puo operare sulla richiesta di trascrizioni o caricamento delle immagini attrraverso il tasto
Hub Manager
i vari utenti registrati con i vari ruoli sono presenti nel database altrimenti
potete registrarvi dall applicazione direttamene ma sarete considerati utenti normali all inizio
il menu ricerchera le opere presenti nel databaase e potrete accedere alle pagine relative di quell opera
e visualizzare sia la copertina della pagina che la trascrizione.
è necessario scaricare anche il database per far funzionare l app perche e tutto connnesso tramite il jdbc 
mettero a disposizione il package diagram per farvi capire l organizzazione del programma.
